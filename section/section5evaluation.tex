\section{Experiments}\label{section:evaluation}

\subsection{Experimental Setup}

\textbf{Base Model.} We use Llama-2-7B-Chat~\cite{hu2023llama} as the base language model for all experiments. This model provides a strong foundation of general language understanding and instruction-following capability, which we adapt to the clinical domain through our personalization framework.

\textbf{Baselines.} We compare our proposed framework against the following baselines:
\begin{enumerate}
\item \textbf{Base Model (Zero-Shot)}: Llama-2-7B-Chat without any fine-tuning, prompted with the patient query and a generic instruction to respond as a physician.
\item \textbf{Full Fine-Tuning (Aggregated)}: Full-parameter fine-tuning of the base model on the aggregated training data from all physicians, producing a single shared model.
\item \textbf{LoRA (Single Aggregated)}: A single LoRA adapter trained on the aggregated data from all physicians, representing the standard PEFT approach without personalization.
\item \textbf{LoRA-Multi (Individual)}: Individual LoRA adapters trained per physician, each loaded separately at inference time without routing or merging. This serves as the upper bound for personalization quality.
\item \textbf{Ours (LoRA + LAG)}: Our framework with individual LoRA training and LAG routing, without TIES-MERGING.
\item \textbf{Ours (Full System)}: The complete framework with LoRA training, LAG routing, and TIES-MERGING.
\end{enumerate}

\textbf{Evaluation Metrics.} We employ a comprehensive set of automatic and human evaluation metrics:
\begin{itemize}
\item \textbf{BLEU-1/4}: n-gram overlap between generated and reference responses.
\item \textbf{ROUGE-1/L}: recall-oriented n-gram and longest common subsequence matching.
\item \textbf{BERTScore}: semantic similarity measured via contextual embeddings.
\item \textbf{Perplexity}: language modeling quality on held-out physician-specific test data.
\item \textbf{Human Evaluation}: three practicing physicians rate generated responses on 5-point Likert scales for (a) Personalization (stylistic alignment with the target doctor), (b) Clinical Accuracy (medical correctness and appropriateness), and (c) Fluency (grammatical quality and readability). Raters also indicate their overall preference among competing methods in a blind A/B comparison.
\end{itemize}
